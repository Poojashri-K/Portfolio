# CloudOptima

## Description
CloudOptima is a multi-cloud cost optimization platform that compares AWS, Azure, and GCP pricing based on user requirements and recommends the most cost-effective cloud provider.

## Features
- User authentication and dashboard
- Resource selection and workload configuration
- Multi-cloud cost comparison (AWS, Azure, GCP)
- Detailed cost breakdown analysis
- Performance score evaluation
- Savings estimation and analysis
- Cloud provider recommendation
- Workload-based cost optimization

## Technologies Used

- CSS

- C++
- Data Structures & Algorithms

## Data Structures Used

- Structures (Struct)
- Greedy Algorithms

## How to Run
- Clone the repository
- Open the project in VS Code
- Navigate to the frontend folder
- Install dependencies using npm install
- Start the development server using npm run dev
- Open the local host URL in your browser
